# LSAT Mission — PT Room 原型使用教程

> 本文档适用于在本地运行并查看 PT Room 功能原型的所有协作人员。

---

## 一、项目概述

这是 **LSAT Mission PT Room** 功能的交互原型，基于 Next.js 构建，完整实现了以下五步用户流程：

| 步骤 | 页面 | 路径 |
|------|------|------|
| Step 1 | The Lounge Hub | `/the-lounge` |
| Step 2 | Room Detail | `/the-lounge/room/1` |
| Step 3 | Exam Interface | `/the-lounge/room/1/exam` |
| Step 4 | Results & Settlement | `/the-lounge/room/1/results` |

技术栈：Next.js 16 · React 19 · Tailwind CSS v4 · Untitled UI 组件库 · TypeScript

---

## 二、环境要求

运行前请确认本机已安装以下工具：

**Node.js**（必须，版本 ≥ 18）

- 检查是否已安装：打开终端，输入 `node -v`
- 未安装请访问 [https://nodejs.org](https://nodejs.org) 下载 LTS 版本

**npm**（随 Node.js 自动安装，无需额外操作）

## 三、快速启动

### 第一步：打开终端，进入项目目录

**macOS / Linux：**
```bash
cd "uiux测试/3.17 pt room测试/lsat-mission-nextjs"
```

**Windows（PowerShell）：**
```powershell
cd "uiux测试\3.17 pt room测试\lsat-mission-nextjs"
```

> 提示：如果你把这个文件夹放在其他位置，请替换为实际路径。

---

### 第二步：安装依赖（首次运行时必须执行）

```bash
npm install
```

安装过程大约需要 1–3 分钟，看到以下输出即表示成功：

```
added 680 packages in 1m
```

---

### 第三步：启动开发服务器

```bash
npm run dev
```

看到以下输出即代表启动成功：

```
▲ Next.js 16.1.7
- Local: http://localhost:3000
✓ Ready in 1.2s
```

---

### 第四步：在浏览器中打开

打开任意浏览器（推荐 Chrome），访问：

```
http://localhost:3000/the-lounge
```

## 四、页面功能说明

### Step 1 — The Lounge Hub
**路径：** `http://localhost:3000/the-lounge`

- 顶部 Hero 区：PT Room 功能入口介绍
- 导航 Tabs：PT Rooms / For You / Q Strategies
- 四个横向滚动卡片列表：
  - **Active PT Rooms**：当前开放/进行中的练习房间
  - **Attack Requests**：社区答题请求
  - **For You**：个性化内容推荐
  - **Q Strategies**：解题策略帖
- **Create Room 按钮**：点击后弹出创建房间 Modal
  - 可设置：活动类型、标题、描述、开始时间（即时/预约）、人数上限、时间限制

---

### Step 2 — Room Detail
**路径：** `http://localhost:3000/the-lounge/room/1`

- 房间标题、状态徽章（Live / Open / Scheduled）
- 可展开的房间描述
- 成员列表：头像、平均分、备考水平、Ready / Waiting 状态
- 底部 Sticky **Join Room** 按钮
- 加入后底部右侧出现**浮动气泡**：
  - 收起状态：成员头像叠加显示
  - 展开状态：成员列表 + Ready 切换 + 房间聊天 + Start 按钮
  - Start 按钮在 ≥2 名成员 Ready 后启用
  - 点击 Start → 跳转考试页

---

### Step 3 — Exam Interface
**路径：** `http://localhost:3000/the-lounge/room/1/exam`

- **倒计时**：35:00 开始，剩余 5 分钟变红
- **顶部成员进度面板**（可折叠）：显示每位成员当前做到第几题
- **左侧题号导航**：四种状态 — 已答（黑）/ 标记（橙边）/ 当前（紫背景）/ 未答（白）
- **答题区工具栏**：
  - Highlight 模式：划选文本高亮
  - Eliminate 模式：点击选项打删除线（`cursor-crosshair` 光标提示）
  - Scratch Pad：右下角浮动便签（可清空）
  - Font Size：四档调节（S/M/L/XL）
- **底部导航栏**：上一题 / 下一题 + 进度条，最后一题显示 "Finish →"
- 点击 Finish → 跳转成绩结果页

---

### Step 4 — Results & Settlement
**路径：** `http://localhost:3000/the-lounge/room/1/results`

- **顶部黑色 Voice Bar**（固定显示，64px 高）：
  - 房间名称 + 每位成员的语音状态（Speaking / Listening）
  - 绿点 = 正在发言，Unmute 按钮切换麦克风
- **左栏（固定宽度）**：
  - Your Score 卡：scaled score、百分位、raw score、各 section 进度条
  - Leaderboard：房间成员按分数排名，相对进度条，"You" 用品牌紫色高亮
  - Save to Error Log 按钮
- **右栏（弹性宽度）**：
  - Section 筛选 Tabs：All / LR I / LR II / LG / RC
  - **Dot Grid**：每点代表一道题，三种颜色：黑=正确 / 红=错误 / 灰=未作答；hover 显示 tooltip
  - **题目展开行**：点击展开显示题干、房间所有成员的选择、以及 Discuss 按钮
    - 错题 Discuss 按钮为红色强调样式
    - 点击 Discuss → 触发右滑 Discussion Panel
- **Discussion Panel（右滑浮层）**：
  - 顶部：完整题目文本，含考试标记（高亮 / 删除线 / ✓ You / ✗ You badge）
  - 底部左侧（57%）：My Notes 白板，可自由输入笔记
  - 底部右侧（43%）：Live Transcript 对话记录，带说话人分辨和波形动效
  - Copy Transcript 按钮：将对话记录 append 到笔记

---

## 五、注意事项

- 本原型所有数据均为**模拟数据**，不连接真实后端
- 计时器、聊天消息、成员状态均为前端仿真
- 语音通话 / Transcript 功能在原型中用静态数据模拟；生产版本将基于 WebRTC + Whisper STT
- 建议在**桌面浏览器**（Chrome / Safari）中查看，移动端布局尚未适配
